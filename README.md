> A powerful, **completely FREE** AI coding assistant powered by Groq's Llama 3.3 70B model. Get instant help with programming, algorithms, debugging, file analysis, and more!

## ✨ Features

| Feature | Description |
|---------|-------------|
| 🚀 **Lightning Fast** | Responses in 1-2 seconds |
| 💰 **100% Free** | No credit card required, 30 requests/minute |
| 📁 **File Upload** | Upload documents, images, or code files |
| 💾 **Chat History** | All conversations saved with meaningful titles |
| 🎤 **Voice Input** | Works on Chrome, Firefox, Edge, Brave |
| 📋 **Copy Code** | One-click copy for all code blocks |
| 📌 **Pin Chats** | Pin important conversations to the top |
| 🔍 **Search Chats** | Quickly find past conversations |
| 🖱️ **Right-Click Menu** | Pin, rename, delete, or open chats |
| 🎨 **Modern UI** | Clean, responsive, full-screen design |
| 🔒 **Secure** | API keys stored safely in .env |

## 🛠️ Tech Stack

- **Backend**: Node.js + Express
- **AI**: Groq API (Llama 3.3 70B)
- **Frontend**: HTML5, CSS3, JavaScript
- **Storage**: LocalStorage (browser)
- **Icons**: Font Awesome 6
- **Voice**: Web Speech API (cross-browser)

## 📋 Prerequisites

- Node.js (v18 or higher)
- Groq API key ([Get free key](https://console.groq.com))
